Assignment 1 | Comparing catagories

Section 1: Barchart

This plot uses a straightforward bar chart to show how common the top tree species group by cities.

The bar chart provides an interactive way to explore the tree species counts in different American cities. Users can select specific cities to view, and the chart displays the count of tree species, including some type of trees, in the selected cities. This visualization allows users to compare tree species counts across various urban areas in America and gain insights into the distribution of these trees.

Section 2: Stacked Barchart

This plot uses a straightforward bar chart to show how common the top tree species group by cities.

This stacked bar chart visualizes tree species counts in five American cities: New York, Los Angeles, Boston, Houston, and Irvine. The selected tree species for comparison include Platanus acerifolia, Pyrus calleryana, and Magnolia grandiflora. The chart's x-axis represents the count of trees, providing a clear view of the distribution of each tree species across the selected cities. The y-axis displays the city names, allowing for easy comparison of tree species counts between different urban areas. This interactive visualization provides valuable insights into the variation of these tree species within each city.

Section 3: HeatMap

This plot uses a straightforward bar chart to show how common the top tree species group by cities.

In this heatmap chart, the color intensity represents the tree species count in various cities. Darker colors indicate a higher count of tree species, while lighter colors correspond to fewer species. The x-axis displays the tree species names, while the y-axis shows the city names, providing a visual representation of tree diversity across different cities.
